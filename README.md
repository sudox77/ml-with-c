# ML from Scratch in C

A collection of simple Machine Learning projects implemented in pure C from scratch.

### Projects

- [01-house-price-predictor](./01-house-price-predictor) - Linear Regression to predict house prices
- [02-decision-tree](./02-decision-tree) - Simple rule-based classifier for child/adult classification

### Goals
- Learn Machine Learning concepts by implementing them in C
- No external libraries (only standard C)

### How to Run

**House Price Predictor**
```bash
cd projects/01-house-price-predictor
make
./house_price
```

**Decision Tree**
```bash
cd projects/02-decision-tree
make
./decision_tree
